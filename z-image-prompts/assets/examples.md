# Example Prompts

Reference patterns from tested prompts. Note the structure: [Medium/Style] + [Subject] + [Clothing] + [Action/Pose] + [Setting] + [Camera/Lighting]

## Key Patterns

1. **Lead with medium**: "This is a [style] photograph/render of..."
2. **Specify subject**: age, ethnicity, gender, body type, face shape
3. **Detail features**: hair style, facial hair, skin tone
4. **Describe clothing**: material, fit, distinctive features
5. **Add environment**: brief location context
6. **Include technical**: lighting quality, camera characteristics
